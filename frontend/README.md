# Frontend

Aplicacao web do sistema de gestao escolar em `Next.js`.

## Referencias obrigatorias

- Documentacao central: `../DOCUMENTACAO/AI`
- Regras para agentes: `../AGENTS.md`

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Observacoes

Antes de qualquer alteracao estrutural, consulte a documentacao central para manter aderencia a:

- multi-tenant por `schoolId`
- isolamento entre escolas
- regras de autenticacao e RBAC
- padroes de integracao com o backend
